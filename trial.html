Image Compression Techniques

This project focuses on various image compression techniques such as PCA (Principal Component Analysis), SVD (Singular Value Decomposition), and Huffman Coding. These techniques are crucial in reducing image file sizes while preserving quality.

PCA (Principal Component Analysis)

PCA is a statistical technique that reduces the dimensionality of the data by transforming it into a set of orthogonal components. It's commonly used in lossy compression where some data loss is acceptable, but it effectively captures the essential features of the image.

SVD (Singular Value Decomposition)

Huffman encoding

Huffman Coding is a lossless compression technique based on the frequency of occurrence of pixels. It assigns shorter codes to more frequent pixel values, ensuring that no data is lost during the compression process.
